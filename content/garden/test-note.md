# Welcome to the Digital Garden

This is a test note to verify that the Obsidian integration is working correctly. The digital garden is a place for thoughts in various stages of growth.

## Features Implemented

- **Wikilinks**: You can link between notes using `[[note-name]]` syntax
- **Backlinks**: Notes automatically show which other notes link to them
- **Tags**: Organize notes with tags for easy filtering
- **Categories**: Essays, Notes, and Projects are organized separately

## What Makes a Digital Garden?

Unlike a blog, which presents finished work in reverse chronological order, a digital garden is:

1. **Exploratory** - Ideas can be incomplete and evolving
2. **Connected** - Notes link to each other, forming a web of knowledge
3. **Growing** - Content is continuously updated and refined

## Next Steps

Try creating your own notes! Just add frontmatter with `published: true` and the garden will automatically include them.

> [!note] Important
> Remember to set `published: false` for notes that aren't ready to share yet.
